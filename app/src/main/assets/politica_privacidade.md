## 1. Coleta de Informações

### 1.1 Informações Fornecidas pelo Usuário
- Nome completo
- Endereço de e-mail
- Foto de perfil (quando fornecida)
- Credenciais de login (armazenadas de forma segura)
- Preferências do usuário

### 1.2 Informações Coletadas Automaticamente
- Endereço IP
- Tipo e versão do dispositivo
- Sistema operacional
- Identificadores de dispositivo
- Dados de uso do aplicativo
- Dados de localização aproximada (quando autorizado)

### 1.3 Fontes de Dados
As informações podem ser obtidas:
- Diretamente do usuário
- De serviços de terceiros (Google, Firebase, APIs externas)
- De interações dentro do app

## 2. Uso das Informações
As informações coletadas são utilizadas para:
- Autenticar usuários (via Firebase Auth, Google Play Services)
- Fornecer funcionalidades essenciais do app
- Personalizar a experiência do usuário
- Realizar verificações de segurança (Recaptcha, autenticação biométrica)
- Armazenar dados localmente usando Room
- Comunicar-se com APIs externas via Retrofit/OkHttp
- Carregar e exibir imagens com Coil
- Enviar notificações e mensagens relevantes
- Melhorar a performance e segurança do app

## 3. Compartilhamento de Informações
Não compartilhamos suas informações pessoais com terceiros, exceto:
- Quando necessário para cumprir exigências legais
- Para prestação de serviços essenciais através de parceiros (como autenticação do Google ou Firebase)
- Em caso de fusão, aquisição ou venda de ativos, mediante aviso prévio

## 4. Armazenamento e Segurança
- Dados sensíveis são criptografados durante a transmissão (HTTPS) e no armazenamento local.
- O Room Database é protegido por mecanismos internos do Android.
- Tokens e credenciais de acesso são armazenados de forma segura.
- Utilizamos práticas recomendadas de segurança, incluindo autenticação biométrica opcional.
